Development Structure
---------------------

+-Mechanisms
  +-Source
    +-Mechanisms
    +-Tests

The following command will clone the solution:

> git clone https://github.com/chrisoldwood/Mechanisms.git Mechanisms

Command Line Builds
-------------------

To perform a build for .Net 3.5:

> .\Build

For .Net Core use:

> dotnet build Mechanisms.net20.slnx

Tests
-----

You can run the .Net 3.5 version unit tests with:

> Source\Tests\bin\Debug\Tests.exe

For .Net Core use:

> dotnet run --project Source\Tests\Tests.net20.csproj
